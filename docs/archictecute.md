# Non-Goals
This project is NOT intended to:
- Execute stock trades automatically
- Manage investment portfolios
- Guarantee investment returns
- Replace professional financial advisors
- Make decisions without supporting evidence

The system is designed to generate investment research,
analysis, and recommendations based on available data.
